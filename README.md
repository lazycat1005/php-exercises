# PHP-EXERCISES

PHP 練習題，使用 jQuery1.12.4、SCSS、HTML、PHP

## 結構說明

```
PHP-EXERCISES/
├── css/                       # for主畫面樣式
├── exercises/                 # 各種主題練習範例集合
│   ├── 08_temperature-unit-conversion/
│   │   ├── ajax/              # 使用 AJAX 方式實作的版本
│   │   ├── php-version/       # 純 PHP 實作版本
│   │   └── README.md          # 題目解說
│   └── 47_telephone-bill-calculation/
│       └──
├── index.php                  # 首頁
└── README.md                  # 本說明文件
```

## 備註

- 每個練習項目以編號命名，方便區分與索引。
- 若有不同實作方式（如純 PHP vs AJAX），會分開放置於對應子資料夾中。
